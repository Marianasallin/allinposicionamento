## ❓ Dúvidas Frequentes

**"E se o cliente esquecer a senha?"**
- A senha é a data de nascimento dele, então ele provavelmente lembra
- Se esquecer, você pode atualizar a data de nascimento no admin

**"Como adiciono um novo cliente ao portal?"**
- Vai em Clientes → Clique no cliente → Atualize a data de nascimento (birthday)
- Pronto! Cliente pode usar data de nascimento como senha

**"Posso mudar a senha do cliente?"**
- Sim, alterando a data de nascimento (birthday) no perfil do cliente no admin
- A nova senha será a data atualizada

**"O cliente pode mudar sua própria senha?"**
- Atualmente não (funcionalidade futura)
- Por enquanto, você gerencia as senhas

**"O portal está seguro?"**
- Sim! Autenticação obrigatória, dados isolados, sessão segura
- Mais seguro que compartilhar PDFs por email 😉
