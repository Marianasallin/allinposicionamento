## 🔑 Explicando a Senha Inicial

A primeira senha de cada cliente é **sua data de nascimento no formato DD/MM/AAAA**. 

**Por quê?**
- É fácil de lembrar (é um dado pessoal que o cliente já sabe)
- Diferencia de uma senha aleatória
- Cria um primeiro acesso suave

**Exemplo:**
- Cliente nascido em **25 de dezembro de 1988**
- Senha será: **25/12/1988**

## 🎯 Passo a Passo: Como Compartilhar

### **Passo 1: Coletar Informações**
Antes de enviar, certifique-se de que tem:
- [ ] Email correto do cliente (formato: primeiro.sobrenome@allinposicionamento.com.br)
- [ ] Data de nascimento do cliente (para formular a senha)
- [ ] Email pessoal ou corporativo para envio

### **Passo 2: Preparar a Mensagem**
Use o template de email acima e preencha:
- `[NOME DO CLIENTE]` → Nome do cliente
- `[CLIENTE.EMAIL@allinposicionamento.com.br]` → Email corporativo
- `DD/MM/AAAA` → Data de nascimento

### **Passo 3: Enviar**
Envie um email ou mensagem WhatsApp com:
1. Link do portal: `https://allinposicionamento.com.br/#portal`
2. Email de acesso
3. Primeira senha (data de nascimento)
4. Explicação do que ele encontra lá

### **Passo 4: Acompanhar**
- Pergunte ao cliente se conseguiu acessar
- Esclareça dúvidas sobre como usar
- Recolha feedback sobre a experiência

## 🔧 Se o Cliente Esquecer a Senha

**Cenário:** Cliente não se lembra de sua data de nascimento ou digita errado

**Solução:**
1. Peça ao cliente para confirmar a data de nascimento
2. Indique o formato correto: DD/MM/AAAA
3. Tente novamente

**Se ainda não funcionar:**
No admin (seu sistema), você pode:
1. Ir para "Clientes"
2. Procurar pelo cliente
3. Verificar/atualizar o campo "Data de Nascimento"
4. Enviar a senha corrigida para o cliente

## 🛡️ Segurança - O Que Explicar ao Cliente

Se o cliente perguntar sobre segurança:

**É seguro?**
✅ Sim! O portal é seguro porque:
- Usa autenticação por email + senha
- Cada cliente vê apenas seus próprios dados
- Dados são armazenados de forma segura
- Sessão se encerra automaticamente quando fecha a aba

**O que o cliente pode fazer?**
- Visualizar: Entregas, resultados, encontros, relatórios
- Não pode: Editar dados, acessar dados de outros clientes

**E se outro usar meu login?**
- O cliente deve não compartilhar suas credenciais
- Se suspeitar de acesso não autorizado, deve avisar Mari imediatamente

## 📞 Suporte ao Cliente

Se o cliente tiver dúvidas sobre o portal:

### **Dúvidas Comuns:**

**"Não consigo acessar"**
- [ ] Confirme que o email está correto
- [ ] Peça para tentar a data de nascimento no formato DD/MM/AAAA
- [ ] Sugira limpar cookies/cache do navegador
- [ ] Peça para tentar em outro navegador

**"Esqueci minha senha"**
- [ ] Peça para usar novamente a data de nascimento
- [ ] Se não lembrar, você pode atualizar no admin

**"Por que vejo X dados?"**
- Dados mostrados são baseados no que foi registrado no sistema admin
- Se algo está faltando, você pode atualizar no admin

**"Como faço para...?"**
- Solicitar reunião: Clique no botão "Solicitar reunião" no portal
- Enviar documento: Clique em "Enviar documento"
- Ver histórico completo: Clique em "Reuniões" ou "Relatórios"
